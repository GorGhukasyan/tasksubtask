# To-Do List Web Application

A simple and responsive To-Do List application built using **HTML, CSS, and Vanilla JavaScript**.

The application allows users to manage their tasks and subtasks efficiently while keeping data saved in the browser using **localStorage**.

---

## Features

- Add new tasks
- Add optional subtasks
- Mark tasks and subtasks as completed
- Automatically sync completion between tasks and subtasks
- Delete tasks or subtasks
- Persistent data using **localStorage**
- Responsive design for mobile and desktop
- Built using **pure JavaScript (ES6+)** without any frameworks

---

The application automatically keeps tasks and subtasks synchronized:

- If **all subtasks are completed**, the parent task is automatically marked as completed.
- If the **parent task is marked as completed**, all subtasks will also be marked as completed.

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
